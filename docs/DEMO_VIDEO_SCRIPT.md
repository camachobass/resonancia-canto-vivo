## Optional 15-second OpenAI insert

From Air, create any musical idea and select **Awaken the world**. Wait for the source label, then alternate between **My song** and **Echo's variation**.

Narration: “OpenAI does one job with clear learning value: it transforms constrained musical choices into concise feedback and a different, valid composition the player can hear. Failures fall back to a visibly labeled curated response.”

## Recording rules

- Hide bookmarks and unrelated tabs.
- Keep cursor movement calm and wait long enough for each chord to sound.
- Do not show the Vercel or OpenAI dashboards, environment variables, or any secret.
- Use the app's own audio; do not add copyrighted music.
- End with the production URL visible.
